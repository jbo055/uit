Welcome to my app!

I have designed this app for an assignment in webdesign.

What is the assignment?

The assignment is to create a simple Flask app using CSS and HTML.

The app should have a page where I present myself, and a page where I explain how I understand Python and Flask. Here I should also tell if I have experience with Flask or other web-frameworks before.

How have I solved the assignment?

I have created a simple app consisting of three pages. A main page, a page where I present myself, and a page where I explain what Flask is.

I have chosen to create a simple design, with minimal colors. This is because I wanted to try out layouts before I focus more on the design and making it look cute.

What did I learn?

I have learned a lot of HTML, and learned how easy it can be to ruin the whole structure, by just using the wrong tags. I have also learned a lot about CSS, and how to use classes and IDs.
